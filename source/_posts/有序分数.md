# 有序分数

```
给定一个整数 N
，请你求出所有分母小于或等于 N
，大小在 [0,1]
 范围内的最简分数，并按从小到大顺序依次输出。

例如，当 N=5
 时，所有满足条件的分数按顺序依次为：

01,15,14,13,25,12,35,23,34,45,11
输入格式
共一行，包含一个整数 N
。

输出格式
按照从小到大的顺序，输出所有满足条件的分数。

每个分数占一行，格式为 a/b
，其中 a
 为分子， b
 为分母。

数据范围
1≤N≤160
输入样例：
5
输出样例：
0/1
1/5
1/4
1/3
2/5
1/2
3/5
2/3
3/4
4/5
1/1
```

题解：



```
在0/1 到 1/1的范围内。有一个中间数 1/2
假设0 = a（左边界分子），1=b（左边界分母） ,1=c（右边界分子） ,1=d（右边界分母）
转换成 a/b c/d
1/2 即可看成是
a+c 
———
b+d

先看左边
0/1 到 1/2之间 又有 1/3作为中间有理数
1/2作为右边界代替了1/1
即
a		a+c
-   ->  ---
b		b+d

再看右边
1/2到1/1之间，又有 2/3作为中间有理数
1/2作为左边界代替了0/1
即
a+c		c
---	->	-
b+d		d
```



```
#include<iostream>
using namespace std;
int n;

void dfs(int a,int b,int c,int d){
    if(b+d >n) return;
    dfs(a,b,a+c,b+d);
    cout << a+c << "/" << b+d << endl;
    dfs(a+c,b+d,c,d);
    
}

int main(){
    cin >> n;
    cout << "0/1" <<endl;
    dfs(0,1,1,1);
    cout << "1/1";
    return 0;
}
```
